#### 第一个Golang api 项目
写在前面，由于没有开发过web程序，这是第一次开发类似的功能，很多东西都不会。
例如怎么分包都不清楚，项目中会有很多不合理的地方。将项目放在github上，记录自己的成长。
1. 基本架构， 原生go+mysql实现api服务，至于其他的，后面慢慢补充。
2. 将项目中用到的隐私信息通过json文件配置，通过加载本地的json，获取相关的配置信息，在
上传代码时，可以保护到自己的数据库等其他信息。
